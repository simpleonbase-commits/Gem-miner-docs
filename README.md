# Gem Miner

> *Mine deep. Earn real. Survive.*

Gem Miner is a real-time, on-chain mining game built on Base. Players descend into a procedurally generated underground world, mining progressively rarer ores to earn **$GEM** — an ERC-20 token native to Base mainnet.

Every ore mined, every upgrade purchased, and every goblin killed is connected to a real wallet. No custodians. No fake currency.

🎮 Play now at **[gemminer.app](https://gemminer.app)**
